# GGPredict Players

Table displays searchable, paginated list of CSGO players

## Installation

Use npm or similar to install dependencies:

```bash
npm install
```

Run dev:

```bash
npm run dev
```

## Stack used

-   Vite + React + TS 💪
-   Tailwind + DaisyUI 🌼
-   Axios, React Query ❤
-   Eslint + Prettier 🧹
